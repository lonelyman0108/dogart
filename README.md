# 🐶 舔狗的艺术

## 构建

由于使用 element-Plus 的按需自动加载功能，需要先安装相关依赖

```
 yarn add -D unplugin-vue-components unplugin-auto-import
```

开启调试

```
 yarn run dev
 ```
